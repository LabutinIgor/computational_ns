1. При увеличении I_{inj} больше некоторого порога у V_m значительный пик, при меньшей амплитуде I_{inj} V_m не сильно меняется.
Чем больше I_{inj}, тем к большему значению возвращается V_m после пика.
Во второй симуляции, при токе меняющемся каждую миллисекунду, при каких-то значениях токов пики образуются около раза за 20 миллисекунд.
При других значениях (например 60 и 70 nA) пик только один вначале.


2. При меньших токах графики фазовых пространств это спирали. Для больших похоже на циклы.
